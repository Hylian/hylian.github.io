Tokilabs Note

2017.05.01

A small controller for the game Sound Voltex.

PCB designed with Boldport's PCBmodE.

Features

  • ATmega32U4
  • RGB button backlighing using Adafruit Neopixels
  • Rotary encoders
  • Sanwa arcade buttons
  • Laser-cut acrylic enclosure
  • Rubber feet inserts
  • 160 x 92 mm
