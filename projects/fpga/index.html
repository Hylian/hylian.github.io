FPGA on the Web

2015.01.01

CMU Hack18 hackathon project to remotely program and interact with Altera DE2-115 FPGA development boards, via a web server. Students can upload SystemVerilog files to be compiled and loaded onto a devlopment board, and then virtually interact with the board.
